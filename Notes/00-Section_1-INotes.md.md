
### Commands

**Note**: sudo == run this command as root (superuser). If we don't use sudo, we don't have permission to make changes.

1. <span style="color: red;">sudo apt update </span> : Updates the package list (PACKAGE_LIST)
2. <span style="color: red;">sudo apt upgrade -y </span> : Installs the new packages (from the PACKAGE_LIST).
3. <span style="color: red;">sudo apt autoremove -y</span> : Removes packages that are no longer needed ⇨ reduces the attack surface.

**Note**: A package is a set of files, programs, libraries, and configurations that provide and contain everything necessary for software to function.


### Drivers
