# Cyber Lab

Before starting the eJPT certification, I built this small lab on my own to gain a better understanding of Linux hardening fundamentals and the theory behind them. The purpose was to build a solid foundation before moving on to more advanced security and penetration testing practice.

## Topics covered

- Linux system preflight checks
- Package updates and patching
- SSH setup and hardening
- Basic networking and subnet communication concepts

## About this repository

This repository contains the notes and materials I created while working through this lab.
